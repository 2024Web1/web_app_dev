## (おさらい)Dockerとは

- アプリケーションをコンテナとしてパッケージ化し、環境差異を気にせずにどこでも実行できるオープンソースのプラットフォーム
- 本授業ではDockerを使用して、PHP、Apache、MySQLを含む開発環境を構築
- 各サービスを個別のコンテナとして実行し、それらを連携させることで、開発環境を迅速にセットアップ<br><br>
